后台管理系统 EasyUI 的使用

进入官网查看：
http://www.jeasyui.net/

下载-部署到项目

点击下载 看到jQuery 点击下载 进入以后找到下载进行下载

找到文档 开始使用

  • 开始使用 jQuery EasyUI

下载库，并在您的页面中引用 EasyUI CSS 和 JavaScript 文件。

<link rel="stylesheet" type="text/css" href="easyui/themes/default/easyui.css">
<link rel="stylesheet" type="text/css" href="easyui/themes/icon.css">
<script type="text/javascript" src="easyui/jquery-1.7.2.min.js">
</script>
<script type="text/javascript" src="easyui/jquery.easyui.min.js">
</script>

根据你下载的找到要用的css 和 js 放在项目中 然后改动href的路径（改成你存放的路径）

  • 在manong_manager_web中找到配置文件spring-mvc.xml 进行静态资源的映射

    <mvc:resources mapping="/css/**" location="/WEB-INF/css/"/>
<mvc:resources mapping="/js/**" location="/WEB-INF/js/"/>

  • 在manong_manager_web中找到controller 创建IndexController.java

    @Controller
public class IndexController {

    @RequestMapping("/")
    public String requestIndex(String name){
        return "index";
    }
}

  • 找到Easyui文档下的Layout布局

    进行自己想要的效果改动代码

  • 布局设置完成 找到Tree 树
    （树（tree）在网页中以树形结构显示分层数据。它向用户提供展开、折叠、拖拽、编辑和异步加载功能。）
